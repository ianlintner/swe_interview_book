## How to Use

1. **Looking up an algorithm?** → [Algorithm Glossary](algorithms.md) — find it by name, get code + related problems
2. **Need a data structure?** → [Data Structure Glossary](data-structures.md) — see operations, complexity, implementation
3. **Searching for a problem?** → [Problem Index](problems.md) — filter by pattern, difficulty, or data structure tag
4. **Studying a pattern?** → [Pattern Reference](../patterns/index.md) — templates and identification tips

!!! tip "Deep Linking"
    Every glossary entry has a permalink anchor. Use these to link from your notes, flashcards, or study plan. Example: `glossary/algorithms/#binary-search`
